# 143 - Job Application & Candidate Portal

**Live Demo:** [https://one43-1s7b.onrender.com/](https://one43-1s7b.onrender.com/)

## 📖 About The Project
This is a full-stack web application built to handle job postings and candidate applications. It serves as a platform where users can view open positions and apply by submitting their details and uploading their CVs. 

The project is built with a modern Javascript stack (MERN) and is designed with responsive UI and a robust RESTful API backend.

## 🚀 Built With
* **Frontend:** React, Vite, TailwindCSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Hosting/Deployment:** Render

## ✨ Features
* **Job Board:** View available open positions.
* **Application System:** Apply to jobs via a dynamic form.
* **File Uploads:** Secure handling of CV uploads (PDF/Docs) via the backend.
* **Responsive Design:** Optimized for both mobile and desktop screens.
